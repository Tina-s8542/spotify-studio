# Spotify Studio 🎵
A Python-based data analysis and machine learning project built using Spotify music data.

## Overview
Spotify Studio is a modular data science project that loads, cleans, analyzes, visualizes, and models Spotify song data to generate insights and recommendations.
The project follows a complete data science workflow from raw data processing to machine learning and song recommendation.

## Project Structure
```text
spotify_studio/
│
├── data_loader.py       # Load Spotify dataset
├── data_cleaner.py      # Data cleaning and preprocessing
├── data_analyzer.py     # Statistical analysis and feature exploration
├── data_visualizer.py   # Data visualization and charts
├── data_modeler.py      # Machine learning models
├── song.py              # Song data representation
├── song_suggestor.py    # Song recommendation system
│
└── data/
│
└── main.py
```

## Features

- Data loading and preprocessing
- Missing value handling
- Exploratory Data Analysis (EDA)
- Data visualization
- Machine Learning modeling
- Song recommendation system
- Modular and object-oriented design

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Workflow

1. Load Spotify dataset
2. Clean and preprocess data
3. Analyze audio features
4. Visualize trends and relationships
5. Train machine learning models
6. Generate song recommendations

## Dataset

Spotify music dataset containing audio features such as:

- Danceability
- Energy
- Valence
- Loudness
- Tempo
- Popularity
- ...
